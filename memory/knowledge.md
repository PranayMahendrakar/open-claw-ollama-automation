## GPU vs CPU — RECOMMENDATION

For this assistant to talk naturally (like Pranay), GPU is STRONGLY recommended.

CPU (current):   Works. Slow for larger models (~5-30s per reply).
                 OK for: deepseek-r1:1.5b, deepseek-r1:3b

GPU (recommended): Much faster, better quality, can run larger models.
                 Recommended: deepseek-r1:7b with 6GB+ VRAM
                 Excellent:   deepseek-r1:14b with 12GB+ VRAM
                 How to enable: OLLAMA_NUM_GPU=1 ollama serve
                 In GitHub Actions: use self-hosted GPU runner

VERDICT: Use GPU if available. The bigger the model, the more
         Pranay-like the conversations will feel.
